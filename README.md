# Button Animation

Button Animation built using HTML, CSS and JavaScript.

A small, lightweight collection of animated button styles that demonstrate CSS transitions and simple JavaScript interaction.

## Features

- Clean, modern button animations using only CSS transforms and transitions.
- Small JavaScript file for optional interactivity.
- Easy to copy/paste and customize for your projects.

## Technologies

- HTML
- CSS
- JavaScript

## Usage

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Button-Animation.git

2. Open `index.html` in your browser (double-click or serve with a static server):

   - Open the file directly: `index.html`
   - Or use a simple server from the project folder:

     - Python 3: `python -m http.server 8000`
     - Node (http-server): `npx http-server`

3. Customize styles in `style.css` or behaviors in `script.js`.

## Files

- `index.html` — demo markup and example buttons.
- `style.css` — all button styling and animations.
- `script.js` — optional JavaScript for interactivity.
- `Screenshot 2024-06-26 233754.png` — preview image used above.
